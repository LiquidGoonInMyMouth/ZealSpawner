This mod is to be used with SuperBLT!

How to setup this mod?

first, extract the ZealSpawner folder from the ZealSpawnerPD2.zip file, then put it in your mods folder inside your Payday 2 directory!

eg:

game/SteamApps/PAYDAY2/mods

also make sure you have SuperBLT installed and set your keybinds inside the mods keybinds option!


and last of all, have fun with this mod and create cool stuff!
